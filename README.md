# 🎟️ TicketLite

TicketLite es una API inicial de prueba desarrollada en Node.js orientada a la gestión de eventos y administración de información relacionada con clientes, calendarios y eventos.

El proyecto implementa una arquitectura organizada por capas (Modelo – Vista/Rutas – Controlador), permitiendo una estructura mantenible para el manejo de operaciones del sistema.

---

## 📌 Características

- Gestión de clientes.
- Administración de eventos.
- Gestión de calendarios.
- Administración de usuarios administradores.
- Conexión con base de datos MySQL.
- API REST usando Express.
- Configuración mediante variables de entorno.
- Soporte para solicitudes HTTP:
  - GET
  - POST
  - PUT
  - DELETE

---

## 🛠️ Tecnologías utilizadas

- **Node.js**
- **Express.js**
- **MySQL**
- **MySQL2**
- **dotenv**
- **CORS**
- **bcrypt**
